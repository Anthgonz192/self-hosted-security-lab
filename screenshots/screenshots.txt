Screenshots

This folder contains screenshots of my home server setup and services.

The screenshots document the configuration and operation of the services running on my Raspberry Pi server. These include monitoring tools, security services, and self-hosted applications that are part of my personal homelab environment.

Examples of what is shown in this folder:
- Uptime Kuma monitoring dashboard showing service status
- Wazuh security monitoring and log analysis
- Ghost blog running as a self-hosted service
- Docker containers used to manage and deploy services
- Network configuration and service availability

These screenshots are included to visually demonstrate the functionality of the environment described in this repository.
